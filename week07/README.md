# Part 2. More Star Wars

In this task, I continued working on a Star Wars API project from the previous week by extending a previous component that displays film data. I created a new web component which fetches and displays character information using a given API URL and formats it into a card layout. I then modified the film component to loop through the list of character URLs and dynamically render each character using the new component. This allowed the page to display both film details and characters  associated in a structured manner.
