# COMP2110 Week 04

# Part 1. Fake News
In this workshop, I used JavaScript and the DOM to interact with and modify the content of Pet Industry News. In Part 1, I inspected the website using Developer Tools, selected headline elements, and used JavaScript to change individual headlines, update all headlines with a loop, and create a function which automatically modifies the headline. 

# Part 2. A Little Web Application
In Part 2, I created a simple button game using HTML, CSS, and JavaScript to style the buttons and implementing click events in sample.js which change the colours of specific buttons when clicked. Through these activities, I was able to gain experience when working with DOM manipulation, event handling, functions, loops, and interactive web page design.
