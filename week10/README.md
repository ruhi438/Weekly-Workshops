# COMP2110 Week 10

Briefly summarise the work you've done this week here.
Step 1. Define Web Application
Form a group with your nearby classmates, groups of 3 or more are encouraged, but flexibility is allowed if needed. In your group, brainstorm and decide on a project idea of your choice. As the delivery of this step, write a brief project description that answers:

What problem does the app solve? Suggested outift combinations (Pictures of items already in closet) based on the weather and occassion

Who are the target users? People who need ideas for outift combinations and need an idea of how to dress for the weather.

What are the key features?

image upload of items and saving of them
categorizing and naming item (light, heavy, warm, bright, fancy, casual, top, bottoms, shoes etc.)
output of the images put together to form an outfit
arrows (left and right) to swap through pieces
Step 2. Analysis Web Application
Then, for your chosen project, perform a basic requirements analysis in a group:

At least 3 core functional requirements (what the app must do)

The user SHALL be able to upload and save outift pieces
The user SHALL be able to add tags to categorise outift pieces
The user SHALL be able to request an outfit based on tags and a weather API request
The user SHALL be able to swap through filtered outift pieces.
At least 2 non-functional requirements (such as performance, security, usability)

Weather API requests SHALL occur once when application has first loaded with temperature stored for later processing.
Loading processes SHALL be displayed to the user (loading wheel)
Pictures SHALL be stored securely
